# Padrão Ouro 2026 — versão corrigida para GitHub Pages

Esta versão usa todos os arquivos na raiz do repositório, evitando a perda da
estrutura de pastas durante o upload pelo navegador.

## Publicação
1. Extraia o ZIP.
2. Envie todos os arquivos extraídos para a raiz do repositório `padrao-ouro`.
3. Confirme a substituição dos arquivos com nomes já existentes.
4. Aguarde a nova publicação do GitHub Pages.

https://altaleite.github.io/padrao-ouro/
