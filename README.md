# Padrão Ouro 2026 - Revisão Institucional V10

Site responsivo e e-book digital do Padrão Ouro de Criação de Bezerras e Novilhas Leiteiras.

## Principais recursos

- nove capítulos técnicos;
- pesquisa e navegação interna;
- controle de tamanho da fonte;
- compartilhamento de capítulos;
- instalação como aplicativo (PWA);
- funcionamento offline dos capítulos e imagens essenciais;
- botões PT / EN / ES para tradução automática;
- e-book PDF acessível, otimizado e com 60 páginas;
- fotografias padronizadas dos 17 autores e conselheiros.

## Idiomas

O português é a referência técnica oficial. Em navegadores compatíveis, a tradução é realizada no próprio dispositivo. Nos demais navegadores, os botões de inglês e espanhol direcionam para a tradução automática do site. Textos incorporados diretamente em algumas imagens podem permanecer em português.

## Publicação no GitHub Desktop

1. Extraia o ZIP.
2. Copie todos os arquivos para a raiz do repositório `padrao-ouro`.
3. Confirme a substituição dos arquivos existentes.
4. Faça `Commit to main`.
5. Faça `Push origin`.
6. Após a publicação, atualize o navegador com `Ctrl + F5`.
