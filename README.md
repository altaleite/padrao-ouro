## Visualização local

Abra `index.html` no navegador. O projeto não depende de servidor nem de bibliotecas externas.

## Publicação no GitHub Pages

1. Acesse a conta/organização `altaleite`.
2. Crie o repositório público `padrao-ouro`.
3. Envie todo o conteúdo desta pasta para a raiz do repositório.
4. Acesse **Settings → Pages**.
5. Em **Build and deployment**, selecione **Deploy from a branch**.
6. Selecione a branch `main` e a pasta `/ (root)`.
7. Salve.

Endereço planejado:

`https://altaleite.github.io/padrao-ouro/`

## Arquivos principais

- `index.html`: página inicial.
- `leitura.html`: ambiente de leitura.
- `autores.html`: autores e biografias.
- `js/content.js`: conteúdo usado pelo site, inclusive em abertura local.
- `conteudo/conteudo.json`: cópia estruturada do conteúdo.
- `css/style.css`: sistema visual e responsividade.
